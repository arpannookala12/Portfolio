2. Fire Detection & Localization (Jan – Jun 2021)

Fire detection system using Inception V3 on images (fire, smoke, neutral), integrated with IoT for real-time Telegram alerts.

Objective & Approach: Combined CNN-based detection (Inception V3) with a Bolt Wi-Fi module and OpenCV for real-time image analysis. Sent alert messages to a Telegram chatbot when fire detection confidence exceeded a threshold.

Dataset & Augmentation: 3,000 images in Fire, Smoke, and Neutral categories. Achieved 91.05% training accuracy, 87.23% validation accuracy with flipping, zooming, and dropout layers to mitigate overfitting.

Future Scope: Incorporate bounding box detection and GPS for real-time localization in large landscapes.
