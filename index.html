Projects

Pot-in-Pot Refrigerator (Jan – Mar 2020)

Eco-friendly fridge using two clay pots with IoT (Bolt) for temperature logging, plus polynomial regression for predictive modeling.

Objective: Build a sustainable “cooling” mechanism using clay pots. Logged temperature hourly and used polynomial regression to forecast temperature changes. Learned the fundamentals of bridging hardware with analytics.

Fire Detection & Localization (Jan – Jun 2021)

Fire detection system using Inception V3 on images of fire, smoke, neutral, integrated with IoT for real-time Telegram alerts.

Approach: Combined CNN-based detection (Inception V3) with Bolt Wi-Fi module for hardware alerts. Achieved ~87% validation accuracy. Future expansions include bounding box detection and GPS for localization.
